# 命名规格

注释多固然好，从根本上解决这个问题，变量名，方法名就直接让人知道在干啥了

# 不要使用 item list i data 等抽象词汇...

例如：获取用户信息，并添加索引

// bad
const data = list.map((item, i) => ({...item, i}))

// good
const usersWithIndex = users.map((user, index) => ({...user, index}))
